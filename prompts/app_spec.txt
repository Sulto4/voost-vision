# Voost Vision - Company Website

## Overview

Modern, bilingual (RO/EN) website for Voost Vision, a Romanian full-stack web development agency helping businesses achieve 2026-standard digital presence.

## Tech Stack

- **Frontend**: React 18 + TypeScript + Vite + Tailwind CSS
- **Backend**: Supabase (Auth, Database, Edge Functions, Storage)
- **Styling**: Tailwind CSS + custom design system
- **i18n**: react-i18next for RO/EN
- **Hosting**: Netlify (frontend) + Supabase (backend)

## Brand

- **Company**: Voost Vision
- **Tagline**: "Viziune digitală pentru afacerea ta" / "Digital vision for your business"
- **Colors**: To be defined (modern, professional - suggest dark theme with accent color)

## Database Schema (Supabase)

### Tables

```sql
-- Portfolio projects
CREATE TABLE projects (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  title_ro TEXT NOT NULL,
  title_en TEXT NOT NULL,
  description_ro TEXT,
  description_en TEXT,
  category TEXT NOT NULL, -- 'web', 'app', 'mobile', 'design'
  tech_stack TEXT[], -- ['React', 'Supabase', 'Tailwind']
  thumbnail_url TEXT,
  images TEXT[], -- array of image URLs
  live_url TEXT,
  featured BOOLEAN DEFAULT false,
  created_at TIMESTAMPTZ DEFAULT NOW(),
  published BOOLEAN DEFAULT true
);

-- Blog articles
CREATE TABLE articles (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  slug TEXT UNIQUE NOT NULL,
  title_ro TEXT NOT NULL,
  title_en TEXT NOT NULL,
  excerpt_ro TEXT,
  excerpt_en TEXT,
  content_ro TEXT NOT NULL,
  content_en TEXT NOT NULL,
  cover_image TEXT,
  category TEXT,
  tags TEXT[],
  author TEXT DEFAULT 'Voost Vision',
  published BOOLEAN DEFAULT false,
  published_at TIMESTAMPTZ,
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- Booking slots
CREATE TABLE bookings (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  client_name TEXT NOT NULL,
  client_email TEXT NOT NULL,
  company TEXT,
  description TEXT,
  booking_date DATE NOT NULL,
  booking_time TIME NOT NULL,
  duration INTEGER DEFAULT 30, -- minutes
  status TEXT DEFAULT 'pending', -- pending, confirmed, cancelled
  google_calendar_event_id TEXT,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- Contact form submissions
CREATE TABLE contact_submissions (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  name TEXT NOT NULL,
  email TEXT NOT NULL,
  subject TEXT,
  message TEXT NOT NULL,
  read BOOLEAN DEFAULT false,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- Admin users (for CMS)
CREATE TABLE admin_users (
  id UUID PRIMARY KEY REFERENCES auth.users(id),
  email TEXT NOT NULL,
  role TEXT DEFAULT 'editor', -- admin, editor
  created_at TIMESTAMPTZ DEFAULT NOW()
);
```
